## GO/NO-GO Decision

### **NO-GO** — 0 CRITICAL, 20 HIGH

**Blocking Issues (must fix before production deploy):**

1. **Cross-chain same-chain filter (RT-OPT-001)** — Cross-chain detector produces same-chain opportunities that flood the pipeline. 84% of execution engine work is wasted validation. This causes sustained backpressure, DLQ growth, and Redis memory pressure. **Fix: 1-line guard in detector.ts:1441.**

2. **Docker Compose Redis eviction (SA-1Q-009)** — `allkeys-lru` eviction policy in Docker configs. This is the **exact same root cause** as the 2026-03-04 NO-GO incident that destroyed consumer groups and 113K stream entries. Must be `noeviction`.

3. **Infrastructure port divergence (SA-1Q-001/002)** — Triple-port mismatch for execution engine between Fly.io, service-ports.json, and Docker Compose. Will cause binding failures in production.

**Should fix (HIGH but not blocking):**
- Coordinator same-chain pre-filter (RT-OPT-002) — defense-in-depth
- Undocumented env vars: BLOXROUTE_URL, FLASHBOTS_RELAY_URL
- Cross-chain detector shutdown timeout < internal disconnect budget
- P1 memory investigation (301MB vs 47-95MB for peers)
